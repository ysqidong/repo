# MoodBox - 情绪反馈盒子游戏说明

## 文件信息
- **文件名**: `moodbox.py`
- **版本**: v3.0
- **类型**: 终端互动游戏

## 游戏简介
这是一个带有非线性情绪系统、成就系统和时间挑战的互动式控制台游戏。你需要与一个“盒子小人”互动，通过试探和拨开开关来探索它的情绪反应，同时管理耐心值和警觉度。

## 运行环境
- Python 3.6+
- 支持的操作系统: Windows / macOS / Linux
- 终端需支持UTF-8编码（自动检测Vim环境）

## 安装与运行
```bash
# 直接运行
python3 moodbox.py

# 或赋予执行权限后运行
chmod +x moodbox.py
./moodbox.py
```

## 游戏机制

### 核心数值
| 数值 | 范围 | 说明 |
|------|------|------|
| 耐心值 | 0-100 | 拨开开关会消耗，归零则游戏结束。随时间非线性恢复 |
| 警觉度 | 0-100 | 试探会增加警觉度，随时间非线性下降。过高会加剧耐心消耗 |

### 非线性特性
- **耐心恢复**: 正弦波动 + 警觉惩罚 + 低耐心加速恢复 + 随机扰动
- **警觉下降**: 分段速率（高警觉下降慢，低警觉下降快）
- **连续试探**: 每次试探增加额外警觉度加成
- **情绪系统**: 根据当前数值显示不同表情（😊😐😕😠😤😨）

### 操作说明
| 按键 | 动作 | 效果 |
|------|------|------|
| `空格` | 试探盒子 | 增加警觉度，触发对话 |
| `空格`+`空格` (双击) | 拨开开关 | 消耗耐心值，增加拨开计数 |
| `H` | 帮助 | 显示操作说明 |
| `A` | 成就 | 显示已解锁/未解锁成就 |
| `Q` | 退出 | 退出游戏并显示统计 |

### 胜利/失败条件
- **胜利**: 拨开开关累计15次（获得盒子小人信任）
- **失败**: 耐心值 ≤ 0（盒子被玩坏）

## 成就系统

游戏包含10个成就，可在游戏中按 `A` 查看解锁进度。

| 成就名称 | 解锁条件 |
|----------|----------|
| 幽灵玩家 | 全程警觉度不超过30（需至少拨开3次） |
| 受虐狂 | 连续试探10次不拨开关 |
| 闪电侠 | 30秒内拨开开关5次 |
| 耐心大师 | 拨开开关15次不Game Over |
| 好奇宝宝 | 触发所有警觉度对话（低/中低/中/中高/高共5档） |
| 速通玩家 | **1分钟内**让盒子生气（耐心值归零） |
| 禅意大师 | 保持耐心值>90超过1分钟 |
| 混沌使者 | 同时满足耐心<20且警觉>80 |
| 完美主义者 | 不触发任何生气警告完成游戏（胜利） |
| 社交达人 | 试探和拨开次数相等且都>5 |

## 策略提示

### 最优玩法（追求胜利）
1. 耐心值 > 80 时再拨开开关（消耗少）
2. 避免连续试探（警觉度会叠加）
3. 警觉度 > 40 时暂停操作，等待下降
4. 目标：15次拨开，尽量控制试探次数

### 成就攻略
| 成就 | 速通技巧 |
|------|----------|
| 速通玩家 | 连续快速试探+拨开，1分钟内耗尽耐心 |
| 闪电侠 | 耐心值高时连续双击空格，快速完成5次拨开 |
| 禅意大师 | 放置游戏1分钟不动，保持高耐心 |
| 混沌使者 | 先试探到高警觉，再连续拨开压低耐心 |
| 完美主义者 | 全程保持耐心>50且警觉<50，温柔互动 |

## 常见问题

**Q: 在Vim中运行出现乱码？**  
A: 游戏会自动检测Vim环境并切换到兼容模式。如果仍有问题，请使用 `:term` 命令在Vim内置终端中运行。

**Q: 双击空格不灵敏？**  
A: 双击间隔需在0.3秒内，请快速连续按空格。

**Q: 如何重置游戏？**  
A: 重新运行 `python3 moodbox.py` 即可开始新游戏。

**Q: 成就解锁后能保存吗？**  
A: 当前版本成就仅限单次游戏进程，退出后重置。如需持久化可自行添加文件保存功能。
