层序遍历的变体，用 level_size 控制每层结点数量

from collections import deque def levelorder_by_level(root): if not root: return queue = deque([root]) while queue: level_size = len(queue) for _ in range(level_size): node = queue.popleft() print(node.val, end=' ') if node.left: queue.append(node.left) if node.right: queue.append(node.right) print() # 每层结束换行
# 输出示例 1 2 3 4 5
遍历应用 — 求树高

每个节点旁标注以它为根的子树高度

核心思想：后序遍历。先递归求左右子树高度，再取 max + 1

def height(root): if not root: return 0 left = height(root.left) right = height(root.right) return 1 + max(left, right)
逐步推演
height(4) = 1（叶子） height(5) = 1（叶子） height(2) = 1 + max(1, 1) = 2 height(3) = 1（叶子） height(1) = 1 + max(2, 1) = 3

从底层叶子开始，自底向上逐层汇总，最终根的高度就是整棵树的高度。

遍历应用 — 表达式树

表达式树：叶子是操作数，分支结点是运算符。不同的遍历方式对应不同的表达式写法。

表达式：3 * 2 + 5

遍历方式访问顺序结果表达式类型
前序 + → * → 3 → 2 → 5 + * 3 2 5 前缀表达式（波兰式）
中序 3 → * → 2 → + → 5 3 * 2 + 5 中缀表达式
后序 3 → 2 → * → 5 → + 3 2 * 5 + 后缀表达式（逆波兰式）

前缀：运算符在操作数前面，不需要括号即可唯一确定运算顺序。

中缀：日常写法，但需要括号区分优先级（如 (3*2)+5）。

后缀：运算符在操作数后面，编译器/计算器内部常用，可用栈直接求值。

# 后缀 3 2 * 5 + 的栈求值过程 栈: [3] → 遇到 3，入栈 栈: [3, 2] → 遇到 2，入栈 栈: [6] → 遇到 *，弹出 2,3，计算 3*2=6，入栈 栈: [6, 5] → 遇到 5，入栈 栈: [11] → 遇到 +，弹出 5,6，计算 6+5=11，入栈 结果 = 11
遍历应用 — 判断完全二叉树

核心思想：层序遍历，遇到第一个空结点后，后面必须全空。

✓ 完全二叉树（正例）

✗ 非完全二叉树（反例）

步骤推演（正例）
步骤 1：1 入队 步骤 2：1 出队，2 和 3 入队 步骤 3：2 出队，4 和 5 入队 步骤 4：3 出队，无孩子 步骤 5：4 出队 步骤 6：5 出队，遍历完成，是完全二叉树 ✓
Python 实现
from collections import deque def is_complete(root): if not root: return True queue = deque([root]) found_empty = False while queue: node = queue.popleft() if not node: found_empty = True else: if found_empty: return False queue.append(node.left) queue.append(node.right) return True
