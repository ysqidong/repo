# 课程10：快速排序 (Quick Sort)

## 教学目标

1. 理解分而治之（Divide and Conquer）的思想
2. 掌握分区（Partition）操作的原理和实现
3. 理解快速排序的递归过程
4. 掌握时间复杂度分析（平均O(N log N)，最坏O(N²)）
5. 了解快速选择算法（Quickselect）

## 课程结构

### 1. 回顾与引入
- 回顾已学过的排序算法（冒泡、选择、插入）
- 提出问题：是否存在更快的排序算法？

### 2. 分区（Partition）
- 分区的概念：选择轴，将数组分为两部分
- 分区步骤详解（左右指针法）
- 分区代码实现

### 3. 快速排序
- 递归思想：分区 → 递归排序左边 → 递归排序右边
- 基准情形：子数组长度为0或1
- 完整代码实现

### 4. 性能分析
- 分区操作复杂度：O(N)
- 平均情况：O(N log N)
- 最坏情况：O(N²)
- 可视化对比

### 5. 快速选择（Quickselect）
- 问题：找第k小的元素
- 原理：利用分区，只处理一半数组
- 复杂度：平均O(N)

### 6. 作业
- 随机轴优化
- 三路快速排序
- 思考题
- 实战题
